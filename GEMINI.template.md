2. **PRESENT** — List Option A / B / C

> Skip for: small bug fixes, typos, following existing patterns.

## 🚫 SECURITY GUARDRAILS

1. **Hardcode Secrets**: Never write API Key/Password into code. Use `process.env`.
2. **Commit Tokens**: Ensure `.env` is in `.gitignore` before committing.
3. **Delete Database**: Never run `DROP TABLE` / delete DB files without explicit confirmation.
4. **SQL Injection**: Always use Parameterized Queries / ORM.
5. **XSS**: Sanitize input. Use `dompurify` when rendering HTML.
6. **Incident**: Found vulnerability → STOP → REPORT → FIX.
